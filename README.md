## INSTALL JSON-SERVER
```
npm install json-server --prefix .
```
## RUN JSON-SERVER
```
$npx json-server db.json
JSON Server started on PORT :3000
Press CTRL-C to stop
Watching db.json...

( ˶ˆ ᗜ ˆ˵ )

http://localhost:3000/

http://localhost:3000/posts
http://localhost:3000/comments
http://localhost:3000/profile
```
